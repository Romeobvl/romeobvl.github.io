Projet BTS SIO — 1ère année

Chocolate'in

PHP · SQL · HTML5 · CSS3

Site vitrine sécurisé pour un artisan chocolatier fictif, avec barre de recherche, protection contre les injections SQL et XSS, et surlignage des mots-clés.

Description du projet

Chocolate'in est un site vitrine pour une chocolaterie artisanale fictive basée à Bordeaux. Le projet met l'accent sur deux axes techniques : la sécurité web et l'expérience utilisateur.

Le premier défi technique était de protéger le site contre les attaques classiques : injections SQL (via requêtes préparées) et XSS (Cross-Site Scripting, via échappement des entrées utilisateur). Le deuxième challenge consistait à implémenter une barre de recherche intégrée dans la navbar, permettant aux clients de trouver rapidement un chocolat ou une confiserie, avec surlignage en jaune des mots-clés dans les résultats. Le site propose également une navigation par gamme de produits (Produits de saison, Chocolats, Confiseries, Dragées…) et une infolettre mensuelle.
